Request URL Response
Get /movies Return list of all movies.
Get /users/:userName/myMovies Return list of personal top movies.
Get /movies/:title Return data about specific movie
Get /genres/:genre Return data aout genre by name.
Get /directors/:name Return data about director by name.
Post /users Allow new users to register with API, providing data in JSON format.
Put /users/:userName/ Allow users to update info using username as request parameter and info as JSON.
Put /users/:userName/myMovies/:movieId Allow users to add movie to their list of favorites providng both userName and movieID as request parameters.
Delete /movies/:userName/myMovies/:movieId Allow users to remove movie from their list of favorites providng both userName and movieID as request parameters.
Delete /users/:userName Allow user to deregister account providing userName as request parameter.
