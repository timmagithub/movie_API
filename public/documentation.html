Request URL Response
Get /movies Return list of all movies.
Get /users/:userName/myMovies Return list of personal top movies.
Get /movies/:title Return data about specific movie
Get /genres/:genre Return data aout genre by name.
Get /directors/:name Return data about director by name.
Post /users Allow new users to register with API.
Put /users/:userName/:email Allow users to update username and email.
Put /users/:userName/myMovies/:title/:year/:genre/:director Allow users to add movie to their list of favorites.
Delete /movies/:userName/myMovies/:movieName Allow users to remove movie from their list of favorites.
Delete /users/:userName Allow user to deregister account.
